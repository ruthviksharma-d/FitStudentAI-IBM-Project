# 💪 FitStudentAI - IBM Project

## 📌 Overview
FitStudentAI is an AI-powered fitness assistant designed specifically for students. It helps users maintain a healthy lifestyle by providing personalized workout plans, diet suggestions, and activity tracking using intelligent algorithms.

This project is developed as part of the IBM SkillsBuild / Internship program, focusing on applying AI and data-driven solutions to real-world problems.
